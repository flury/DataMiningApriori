# Data Mining Apriori
Data mining Associate Rule using Apriori Algorithm Implementation in Java Spring Framework

Function:
1. Upload dataset
2. Processing data
3. Result (show result in table list)
4. Details (show detail result for each process)

Development stack:
* Java 1.7
* Spring MVC Framework
* Spring Security Framework
* Boostrap 5.0
* Mysql Community Edition
* Apache Tomcat 8.0
* Maven Dependency

Available:
1. Frequently itemset
2. Association Rules
3. Calculate Support
4. Calculate Confidence
